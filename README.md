# Sales Forecasting & Demand Intelligence System

## Project Overview

This project analyzes historical retail sales data and develops a demand forecasting system to support inventory planning and business decision-making.

The project includes:

- Exploratory Data Analysis (EDA)
- Time Series Analysis
- Sales Forecasting
- Anomaly Detection
- Product Demand Segmentation
- Interactive Streamlit Dashboard

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Prophet
- XGBoost
- Scikit-learn
- Streamlit

---

## Forecasting Models

- SARIMA
- Facebook Prophet
- XGBoost

Models were evaluated using:

- MAE
- RMSE
- MAPE

---

## Features

- Sales trend analysis
- Time series decomposition
- Three forecasting models
- Product category forecasting
- Regional forecasting
- Anomaly detection
- Product demand clustering
- Interactive Streamlit dashboard
